# EveryTimeFanc
## Create Automatically short videos with "andate a fanculo" and a time reference (day of week, day time ...)

## Requiremnts

* Python3
* lame
* ffmpeg
* an Apple computer... I use the command "say"

## Usage

```bash
python3 create_videos.py
ls videos/
```
